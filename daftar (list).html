Contoh Tiga Jenis Daftar dalam HTML

1. Daftar Berurutan (Ordered List)

  1. Langkah pertama
  2. Langkah kedua
  3. Langkah ketiga

2. Daftar Tak Berurutan (Unordered List)

  • Buah
  • Sayur
  • Daging

3. Daftar Definisi (Definition List)

HTML
Bahasa markup untuk membuat struktur halaman web.
CSS
Bahasa untuk mendesain dan mengatur tampilan halaman web.
JavaScript
Bahasa pemrograman untuk membuat halaman web menjadi interaktif.
